## 📜 Description

Crafting impactful LinkedIn posts can be time-consuming and inconsistent. This app automates the process using cutting-edge **generative AI techniques**. Whether you're writing about a hackathon, product launch, or personal achievement — this tool generates beautifully structured, platform-optimized content in seconds.

---

## 🧠 Core Features

- ✅ **Few-Shot Learning**: Mimics writing style from handpicked high-performing posts.
- 🎯 **Prompt Engineering**: Dynamic prompts tailored to input like language, tone, topic, and length.
- 🌐 **Language Support**: Supports English & Tanglish (Tamil-English hybrid in English script).
- 📝 **Structured Output**: Generates posts in `WHAT–WHY–HOW–IMPACT` format for maximum clarity and engagement.
- ⚡ **Groq-Enhanced Inference**: Ultra-fast and optimized model performance.
- 🎛️ **Streamlit UI**: Simple, responsive, and user-friendly interface for seamless interaction.

## 🚀 How to Run

```bash
# Clone the repository
git clone  https://github.com/Phoenixarjun/LinkedIn-Post-Generator-Tool
cd LinkedIn-Post-Generator-Tool

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
````
